	thinkNode是一个免费开源的，快速、简单的面向对象的轻量级Node测试框架

#### 适用场景
 - 如果你需要快速建立一些接口来做测试
 - 如果你需要无脑建立一个 http/https 网站

**另外, 如果你是node小白, 想要学习nodeJS, 你可以试阅读这个项目的代码, 注释全中文比较好理解, 这个项目只使用了一些简单的typescript/ES6**
